# 1단계 큰 제목
## 2단계 중간 제목
### 3단계 소제목

**1. 텍스트 강조**
* **굵게**: **내용**
* *기울임*: *내용*
* ~~취소선~~: ~~내용~~

**2. 목록(리스트)**
1. 순서 있는 목록 첫 번째
2. 순서 있는 목록 두 번째

* 순서 없는 목록
    * 들여쓰기로 하위 목록 만들기
    - 대시(-) 기호도 사용 가능

**3. 링크 및 이미지 추가**
* **웹사이트 링크**: [클릭할 글자](https://www.google.com)
* **이미지 삽입**: ![이미지 설명](이미지_파일_경로_또는_URL)
  * 예: ![레나 이미지](./lenna512.bmp)

**4. 코드 블록 (OpenCV 예시)**
문장 중간에 쓰는 `inline code`는 백틱(`) 한 개로 감쌉니다.

여러 줄 코드는 백틱 세 개 뒤에 언어 이름을 적습니다:
```cpp
#include <opencv2/opencv.hpp>
using namespace cv;

int main() {
    Mat img = imread("lenna.bmp");
    return 0;
}

* 별표를 사용한 목록
- 대시를 사용한 목록
+ 플러스 기호를 사용한 목록

    * 한 번 들여쓰기(스페이스 4칸 또는 Tab)를 하면 하위 목록이 된다.
