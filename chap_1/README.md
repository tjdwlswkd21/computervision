**1. 컴퓨터 비전기술이 적용된 제품을 찾아 제품의 기능 동작원리 등을 조사하여 제출하시오**  
컴퓨터 비전기술의 대표적인 예로 스마트폰의 얼굴 인식 잠금 해제 기능이 있다. 
전면 카메라와 적외선 센서를 활용하여 사용자의 얼굴 특징을 추출하고 등록된 데이터와 비교하여 본인 여부를 판단하여 카메라가 얼굴의 눈과 코 그리고 입의 위치와 같은 수만 개의 특징점을 파악하고 이를 수치화된 데이터로 변환한다. 
변환된 데이터가 기존에 저장된 사용자의 얼굴 데이터베이스와 일치하면 즉시 잠금을 해제하며 주변 밝기나 안경 착용 여부에 관계없이 정확하게 인식하기 위해 고도화된 영상 처리 알고리즘이 적용된다.

**2. 영상처리와 컴퓨터 비전의 차이점을 자세히 설명하시오**  
영상처리는 영상 형태의 데이터에 필요에 따른 가공을 거쳐 더 나은 영상을 출력하는 것이고, 컴퓨터 비전은 영상 이미지를 분석하여 필요한 정보를 다양한 형태의 데이터로 추출해내는 작업이다.

**3. CMOS 센서의 원리를 자세히 조사하시오**  
CMOS센서는 빛을 전기 신호로 변환하는 반도체 소자로 데이터 처리가 빠르고 전력 소모가 적은 것이 특징이다.
먼저 센서 표면의 각 픽셀에 있는 포토다이오드에 빛이 부딪히면 광전 효과에 의해 전자가 발생하고, 빛이 강할수록 더 많은 전자가 생성된다.
또한 각 픽셀마다 증폭기가 붙어 있어 발생한 전하가 픽셀 내부에서 즉시 전압 신호로 변환되고 증폭돼 신호를 멀리 보낼 필요가 없어 노이즈 발생이 적고 전력 효율이 높습니다.
증폭된 아날로그 전압 신호는 행과 열단위로 읽혀 센서 내부에 통합된 ADC를 거쳐 디지털 데이터로 변환되는데, 이때 병렬 방식의 데이터 처리로 인해 고속 연사 및 고해상도 영상 촬영에 유리하다.
포토다이오드는 빛의 세기만 감지하고 색상은 구별하지 못 하기 때문에 센서 위에 빨강(R), 초록(G), 파랑(B)으로 구성된 필터를 씌워 각 픽셀이 특정 색상의 빛만 받아들이게 한 후 
출력 시에 이미지 프로세서가 이 정보들을 각 색에 대응시켜 조합해 천연색을 복원시켜 사진을 출력한다.

**4. 영상의 해상도와 이미지 센서의 개수의 관계를 설명하시오**  
영상의 해상도는 영상의 픽셀 수에 의해서 결정되는데, 이미지 센서 하나가 영상의 픽셀 하나를 만들어내므로 이미지 센서가 많을 수록 영상의 해상도가 좋아지고 원본과 같은 이미지를 저장할 수 있다.

**5. 100 X 100 그레이스케일 영상 파일의 크기 바이트 를 계산하라**  
그레이스케일영상의 한 픽셀은 1바이트의 크기를 가지므로 100*100 크기의 경우 10000바이트 -> 10KB 의 크기를 가진다.

**6. 100 X 100 컬러 영상 파일의 크기 바이트 를 계산하라**  
컬러 영상의 경우 한 픽셀당 각각 r, g, b 밝기값을 따로 저장하여 3바이트의 크기를 가지므로 30KB 의 크기를 가진다.
